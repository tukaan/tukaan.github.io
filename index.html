    Tukaan is a cross platform Python toolkit for creating desktop GUI applications

    pip install tukaan

    Powerful. Modern.
    Intuitive.

    With Tukaan you can create amazing GUI apps with ease. Tukaan is intuitive and powerful - the complete opposite of Tkinter, although it was initially designed as an alternative to it.
